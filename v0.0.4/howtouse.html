HOW TO USE

To encipher a message:
1. Set the encipher/decipher knob to ‘C’

2. Reset the letter counter to zero
   a. To do this, push the reset button and turn the reset knob until the counter clicks into position

3. Rearrange all six key wheels in a random order. This ordering will be saved as your External Message Indicator (EMI)
   a. Note: this EMI should only be used once in conjunction with a particular pin and lug setting

4. Select, at random, any letter on the indicating disk and record this letter

5. Encipher the letter from step four 12 times. Remove the resulting tape as it will help you make an Internal Message Indicator (IMI)

6. Reset the letter counter to zero

7. Set the first letter on the tape from step five on the first key wheel. Continue setting these letters to your six key wheels. If a letter is not available on a given wheel, cross it out and move on to the next letter on the tape. These six resulting letters are your IMI.

8. Encipher your message. Note that the letter count should increase with each letter you encipher.
   a. At the end of each word, encipher a ‘Z’.
   b. If you have a word made up of a single letter, spell out its phonetic name (D = delta).
   c. Numbers need to be spelled out (2 = TWO)

9. When you’re done enciphering your message, roll out extra tape on each end.

10. Because it will encipher in five letter groups, if your last group does not have five letters, you need to add as many ‘X’ characters as needed to make it a five-letter group.

11. Before sending your message, you need to add three indicators to the tape.
   a. System indicator: write the letter you enciphered in step 5 twice at the beginning and end of the tape you want to send
   b. Write EMI from step 3. This will need to be broken up into two, three letter groups to obey by the five-letter groupings.
   c. Key list indicator: write the two letters which indicate which chart to use for the pin and lug settings

12. Mix up your key wheel so the IMI is lost.

13. Send your message.


To decipher a message:
1. Set the letter counter to zero

2. Turn the encipher/decipher knob to C

3. Locate the EMI in your message and set the key wheels to this letter combination.

4. Encipher the indicated repeated letter 12 times to be able to determine IMI.

5. Reset the letter counter to zero.

6. Repeat step 7 from the encipher section to set up your IMI.

7. Change the encipher/decipher knob to the D position.

8. Set each letter on the indicating disk to decipher your message.
   a. Disregard the spaces between the five-letter sections
   b. You must write in the X’s at the end of the message because you don’t know if they’re meant to be disregarded or not
   c. All Z’s will come out as blanks

9. When done deciphering, remove the tape, fill in the missing letters that were given as spaces and maybe remove any trailing X’s.

10. Turn your keys to break up the IMI


Setting up the pins and lugs:
You will need the charts which correspond to various pin and lug settings (Signal Operating Instructions)

Pins:
   1. Set all letter pins to the left side

   2. Read down the chart and set the pins as indicated. A letter means the pin goes to the right and a dash means the pin goes to the left. Leftward position denotes ineffective while right position denoted effective.

   3. Repeat this process for all 6 columns which refer to the six key wheels.

Lugs:
  1. The numbers on the left of the chart refer to the 27 drum bars. Each bar has two moveable lugs

  2. The two numbers to the right of this column refer to the left and right lug positions.

  3. 2   3-4 means the left lug on the second bar needs to be set to position 3 and the right lug on the second bar needs to be set to position four.
     a. The positions are indicated by the number plate.

0 denotes that the lug be set to the neutral position
